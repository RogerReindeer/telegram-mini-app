# Зефиркины баоцзы — Mini App reader

## Доступы

- guest — только бесплатные главы (`public`).
- traveler — бесплатные + главы подписки Boosty (`subscriber`, `boosty`, `traveler`).
- keeper — все книги и главы, включая `premium`, `paid`, `early`, `keeper`, `hidden`, если у главы есть ссылка.

Платный текст не передаётся в браузер целиком. Неавторизованный пользователь получает только первые 1–2 предложения и декоративные размытые строки.

## Telegram

1. Добавьте Mini App-бота администратором в обе приватные группы.
2. Узнайте числовые ID групп вида `-100...` и задайте `TRAVELER_CHAT_ID` и `KEEPER_CHAT_ID`.
3. Добавьте `TELEGRAM_BOT_TOKEN`, ссылки-приглашения и `SESSION_SECRET` в Render Environment.
4. Открывайте сайт как Telegram Mini App. Обычный браузер работает в роли гостя.

Ссылки-приглашения `t.me/+...` не являются chat_id и используются только для кнопок перехода.

## Fox

Лисички загружаются из Supabase-таблицы `fox`, синхронизируемой из Excel/Google Sheets. Поддерживаются имена с `.png`, например `fox_side.png`: расширение автоматически удаляется. Прямые ссылки `https://img2.teletype.in/files/...png` используются без преобразования.
